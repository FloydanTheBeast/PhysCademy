Альберт Эйнштейн существенно изменил представления людей об устройстве Вселенной, пространства и времени. Книга "Эволюция физики", написанная создателем специальной и общей теории относительности совместно с известным польским физиком-теоретиком Леопольдом Инфельдом, приближает к пониманию законов, которые управляют материей.
